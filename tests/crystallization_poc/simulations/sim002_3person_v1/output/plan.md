# 协作方案

# 协作构型提炼报告

## 一、协作全景

三位参与者构成一个"多维诊断-干预"系统，共同为一家东南亚制造企业（800人，从OEM代工转型品牌DTC）提供深度转型支持。Lina通过纪录片式影像捕捉组织真实状态，Maya通过组织诊断和现场干预重构信息流动方式，赵维通过系统建模和技术架构设计提供数字化基础设施。三人的工作不是并行交付三份报告，而是形成一个"观察-翻译-建模"的联合认知系统，在三个月内完成诊断并产出可执行的转型路径。这个组合能达成的是：让CEO看见企业的真实运作逻辑（而非报表和会议呈现的版本），让中层和一线感到"转型"不是又一次自上而下的折腾，让数字化系统建立在对真实信息流的理解之上而非技术供应商的标准方案。

## 二、各参与者方案

### **Lina**

- **角色**：组织真实状态的影像记录者与叙事设计师

- **贡献**：
  - 2-3个月的现场跟拍，记录管理层会议、中层协调、一线操作、客户互动等关键场景
  - 产出一个"内部版本"纪录片（完整、未经美化的组织诊断影像，版权归企业）
  - 产出若干阶段性影像片段（每两周3-5分钟），用于三人联合汇报时的"视觉锚点"
  - 在现场通过"隐形观察"建立信任基线（前三周），为Maya的干预和赵维的访谈创造"真实对话"的可能性
  - 拍摄关键转型时刻（如第一次中层工作坊、系统地图讨论会），记录微表情和非正式对话
  - 潜在的"外部版本"案例纪录片（脱敏后用于Lina自身的"真实商业故事"内容品牌，在转型成功后1-2年制作）

- **收益**：
  - 将"纪录片能力"从单纯的内容创作工具变为"组织诊断工具"，验证"真实本身就是内容力"的商业价值
  - 获得一个完整的、可复用的"制造业转型"案例素材库（可用于未来的外部版本创作和方法论展示）
  - 从"接散活"的不稳定模式进入"作为顾问工具的内容创作"这个新商业定位
  - 与Maya和赵维建立长期协作关系的可能性（如果这次项目成功，三人可能形成一个可复用的"联合诊断工作组"）
  - 项目费用约28万人民币（70万总包的40%）

- **成本**：
  - 2-3个月的全职现场时间（一个导演+一个助理）
  - 放弃这段时间内其他可能的品牌内容项目收入
  - 需要接受"前三周看起来什么都没做"的节奏，承受CEO可能的焦虑压力
  - 需要在"隐形观察"和"团队协作"之间找到平衡（例如：第三周汇报时她在场但不说话，这个状态需要她提前接受并向客户解释）
  - 内部版本影像的版权归企业所有，无法直接对外使用

- **依赖**：
  - 依赖Maya在前两周建立组织信任（通过非威胁性的倾听和对话），降低员工对镜头的防御
  - 依赖赵维提前识别"信息流断点"的关键场景，让她知道该重点拍摄哪些节点
  - 依赖CEO在第一次见面时接受她的"测试性问题"（"如果我们拍到的东西显示转型失败的原因是X，您能接受吗？"），确认客户真的想看到真实而非宣传片
  - 依赖三人在每天现场结束后的同步会议，及时交换"今天最意外的观察"，避免她在错误方向上浪费拍摄时间

### **Maya**

- **角色**：组织动力学诊断者与协作流程设计师

- **贡献**：
  - 前两周进行组织"隐形结构"诊断：通过20-30个不同层级的非正式对话，识别真正的意见领袖、信息流断点、非正式权力网络
  - 第三周开始进行"测试性干预"（在相对安全的一对一场景中提出挑战性问题，观察系统反应）
  - 第四周开始"联合干预"（在Lina的镜头记录下，在关键会议或工作坊中做刺激性提问和流程调整）
  - 设计并主持"行动地图工作坊"（第四周或第五周），让中层在白板上画出"我们认为可以改变的路径"，建立ownership
  - 作为三人协作的"协调层"：主持每周五的三人同步会议，做跨领域翻译（把Lina看到的画面、赵维发现的数据断点、她观察到的行为模式，放在同一个框架里理解）
  - 推动决策机制：当三人对节奏有分歧时（如Lina认为还需一周建立信任 vs Maya判断现在应做刺激性干预），通过"把分歧变成可测试的假设"来破局

- **收益**：
  - 将"组织诊断方法论"在一个真实的、复杂的转型场景中完整实践，验证"不做表面功夫"的深度干预是否可行
  - 与Lina和赵维的协作中学习"如何让我的方法论可视化"（Lina的影像）和"如何让组织洞察可技术化"（赵维的系统地图）
  - 突破目前咨询业务的天花板：从"一个顾问服务两个客户"变成"三个顾问深度协作服务一个客户"，测试这种模式是否可以规模化复用
  - 为未来的"AI辅助组织诊断"积累方法论数据（例如：哪种干预方式让场景从🟡变成🟢、哪些隐形结构可以被模式化）
  - 项目费用约24.5万人民币（70万总包的35%）

- **成本**：
  - 第一个月需要全职在现场（每周5天），之后可以每周3-4天
  - 需要在新加坡的咨询公司业务中腾出30-40%时间（约3个月）
  - 承担"协调三人协作"的隐性成本（主持周五会议、处理分歧、推动决策），这部分工作在项目报价中没有单独体现
  - 如果转型失败或CEO中途终止项目，她的时间投入无法被其他客户填补（因为这三个月她拒绝了其他咨询机会）

- **依赖**：
  - 依赖Lina在前三周的"隐形观察"为她创造"真实对话"的基础（如果员工一开始就高度防御，她的倾听和干预都会失效）
  - 依赖赵维的"信息流图"为她提供"可视化的组织结构"（她的隐形结构诊断是定性的、基于对话的，赵维的图是定量的、基于数据的，两者叠加才能形成完整诊断）
  - 依赖CEO在"预期对齐对话"中承诺"给她三样东西：时间、开放度、保护（当中层抵制时你要撑住我们）"
  - 依赖三人在每日同步会议中及时交叉验证信息（例如：某个中层分别跟三人说了不同版本的话，如果不及时发现会导致诊断被"分化"）

### **赵维**

- **角色**：信息流系统建模者与最小可行技术方案设计师

- **贡献**：
  - 第一周进行关键中层和一线的访谈，追踪"一个订单从进来到交付，数据经过了哪些人、哪些系统、在哪里被改写、在哪里丢失"
  - 第二周绘制"现状信息流图"，标注明确的断点和隐藏的依赖关系
  - 第三周制作"可交互的系统状态地图"（网页或Figma原型），作为第一次联合汇报的核心工具
  - 持续维护"诊断地图"（呈现信息流的实然状态），同时管理"观察面板"（中层可以添加卡片，描述他们看到的现象和认为的问题）
  - 第四周或第五周，在行动地图工作坊后，将白板上的中层共识数字化为"行动地图"（开放编辑权限）
  - 定义"做DTC需要追踪的10个最关键数据点"，设计"最小可行基础设施"（可能一开始只是共享表格+自动化脚本）
  - 作为三人协作的"状态机维护者"：管理共享的Notion数据库（记录每天的观察、干预时间点、🟡🟢信号系统的状态变化），每周五输出"本周系统地图更新"
  - 评估技术供应商方案的合理性（如果CEO后续需要上CRM或数据中台，赵维帮他判断50万和500万的方案差在哪里）

- **收益**：
  - 将"分布式系统能力"应用到一个全新领域（组织的信息流），验证"技术系统设计"和"组织协作设计"是否有可迁移的模式
  - 学习"隐性知识如何在组织中流动"（这是他一直想做的"知识捕捉工具"的前置研究）
  - 与Maya和Lina的协作中学习"如何从人的视角理解系统"（Maya的组织洞察）和"如何让系统可被感知"（Lina的影像记录）
  - 积累"三人协作的信息系统"的产品原型（可以脱敏后作为开源工具或未来的SaaS产品）
  - 项目费用约17.5万人民币（70万总包的25%）

- **成本**：
  - 第一个月需要每周3-4天在现场（访谈、看流程、验证假设），之后可以部分远程工作
  - 需要在两周内完成"可交互系统地图"的开发（使用Observable或D3.js）
  - 承担"协调层"和"状态机维护"的隐性工作量（管理共享数据库、每周输出系统地图更新、在Lina和Maya需要实时决策时充当"联合判断"的第三方）
  - 如果项目中期发现"技术部分的工作量超预期"（例如：客户的信息系统比预想的更混乱），他需要在不重新谈价格的前提下投入更多时间（因为对外是"联合服务包"）

- **依赖**：
  - 依赖第一周能见到关键中层和一线（"如果第一周只能见CEO，那第二周才能见中层，我就没时间了"）
  - 依赖Lina拍摄"中层看诊断地图时的反应"（皱眉或点头），帮他判断诊断是否跟中层经验对得上
  - 依赖Maya在第四周的"行动地图工作坊"中引导中层用便利贴画出"可以改变的路径"，他才能把这些内容数字化为行动地图
  - 依赖三人在"半天场景模拟"中提前推演前四周的具体协作画面（"第一周第一天三个人同时出现在客户公司，Lina去哪里？Maya跟谁说话？我在干什么？"），避免现场互相干扰

## 三、协作顺序与节点

### **阶段0：启动前准备（项目开始前1周）**

- **三人场景模拟会议（半天）**：推演前四周的具体协作画面，对齐"谁在什么时候出现在哪里、做什么、如何避免互相干扰"
- **第一次见CEO（三人共同在场）**：
  - 赵维主导沟通，明确介绍三人角色，特别是Lina的"观察者"定位（"她不参与讨论，但会在现场拍摄和观察...她拍的不是宣传片，而是组织的真实状态"）
  - Lina进行"测试性提问"（"如果我们拍到的东西显示转型失败的原因是X，您能接受吗？"）
  - Maya进行"预期对齐对话"（"你需要给我们三样东西：时间、开放度、保护"）
  - 三人用三个不同角度的问题测试CEO是否真的想看到真相（Lina问能否接受负面发现、赵维问能否诊断高层决策层、Maya问CEO本人是否愿意调整行为模式）
- **商业条款确认**：
  - 总价60-80万人民币（最终确定为
