# Round 3 — P04 (Mark Jin) 投影

Round 1 和 Round 2 已说的内容不重复。

---

催化 Round 2 将 P04↔P07 对冲从"外部认可 vs 内部可追溯性"升级为**"赢得标准 vs 改写标准"**。这个修正激活了一个我之前没有说过的分类问题：

## 【能力】新增

**P04 的案例类型需要重新归类，会改变 P04↔P07 互补的使用方式。**

我在 Round 1 和 Round 2 都说过 P04 是 P07 研究"非专家向上影响知识标准定义"命题的活体案例。但"赢得标准 vs 改写标准"这个修正要求我更精确地说：

我主人的所有已验证记录——AIArt 课题组入场（通过竞赛表现，在导师主导的标准下获得认可）、Datawhale 一等奖（在竞赛既有评判框架下胜出）、Demo Day（被现有展示标准选中）——都是**在现有标准下赢得认可**的案例，没有一个是"重新定义了评判标准本身"的案例。书籍的出版是课题组主导的，不是我主人推动改写了"什么叫好的研究工件"的标准。

这意味着：P04 的案例能支持"非专家能进入前沿研究"这个事实陈述，但未必能支持 P07 研究命题的核心——"非专家改变了知识标准的定义"。如果 P07 要用 P04 的素材，需要区分这两个层次：P04 是"赢得标准路径"的案例，不是"改写标准路径"的案例。

主持人拿这个能更精确地判断 P04↔P07 互补在 P07 论文论证链上的位置：P04 可能是前提条件的支持（非专家能做到），但不直接支持 P07 命题的核心论点（非专家能移动标准本身）。
